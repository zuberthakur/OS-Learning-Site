📘 Unit 3: Memory Management

💻 What is Memory Management?

Memory management Operating System ka function hai jo computer ki main memory (RAM) ko manage karta hai. Iska kaam processes ko memory allocate aur deallocate karna hota hai.

🧠 Types of Memory

  • Primary Memory (RAM)
  • Secondary Memory (HDD/SSD)

📦 Memory Allocation Techniques

  • Fixed Partitioning
  • Dynamic Partitioning
  • Paging
  • Segmentation

⚙️ Paging

Paging ek memory management technique hai jisme process ko equal size blocks (pages) me divide kiya jata hai.

🧩 Segmentation

Segmentation me process ko logical segments (like code, data, stack) me divide kiya jata hai.

⚠️ Fragmentation

  • Internal Fragmentation
  • External Fragmentation

🔒 Virtual Memory

Virtual memory ek technique hai jo secondary memory ko RAM ki tarah use karne deti hai.
