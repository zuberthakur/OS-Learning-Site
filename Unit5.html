📘 Unit 5: Security & Protection

🔐 What is OS Security?

Operating System security ka matlab hai system ko unauthorized access, misuse aur attacks se bachana.

🛡️ Goals of Security

  • Confidentiality
  • Integrity
  • Availability

🔑 Authentication

Authentication ek process hai jisme user ki identity verify ki jati hai (password, OTP, biometrics).

👤 Authorization

Authorization decide karta hai ki user ko kya access milega aur kya nahi.

⚠️ Threats to OS

  • Virus
  • Worms
  • Trojan Horse
  • Hacking

🔒 Protection Mechanisms

  • Access Control
  • Firewall
  • Encryption
  • Antivirus Software
