📘 Unit 2: Process Management

💻 What is a Process?

Process ek program ka execution hai. Jab koi program run hota hai, to use process kehte hain. Har process ka apna memory space aur resources hote hain.

🔄 Process States

  • New – Process create ho raha hai
  • Ready – Execution ke liye ready hai
  • Running – CPU use kar raha hai
  • Waiting – Kisi event ka wait kar raha hai
  • Terminated – Process khatam ho gaya

📋 Process Control Block (PCB)

PCB ek data structure hai jo har process ki information store karta hai jaise:

  • Process ID
  • Process State
  • CPU Registers
  • Memory Info

⏱️ CPU Scheduling

CPU scheduling ka matlab hai CPU ko different processes ke beech allocate karna.

📊 Scheduling Algorithms

  • FCFS (First Come First Serve)
  • SJF (Shortest Job First)
  • Round Robin
  • Priority Scheduling

⚠️ Deadlock

Deadlock ek situation hai jisme processes ek dusre ka wait karte rehte hain aur koi bhi execute nahi hota.

🔒 Conditions of Deadlock

  • Mutual Exclusion
  • Hold and Wait
  • No Preemption
  • Circular Wait
